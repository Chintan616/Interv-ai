## 📖 About

**Interv.ai** is an intelligent interview preparation platform that helps job seekers practice and excel in technical interviews. Powered by OpenRouter AI models (like Google Gemini and Qwen), it generates personalized interview questions based on your specific role, experience level, and even the contents of your own resume.

### Why Interv.ai?

- 🤖 **AI-Powered**: Get realistic, challenging interview questions generated by top-tier AI models.
- 📄 **Resume Parsing**: Upload your PDF resume to get grilled specifically on your past projects and skills.
- 🎯 **Personalized**: Questions tailored to your specific role and years of experience.
- 📝 **Interactive**: Answer questions and get instant, detailed AI feedback with code examples.
- 📊 **Track Progress**: Save sessions, pin important questions, and add personal notes.
- 💼 **Role-Specific**: Support for all tech roles (Frontend, Backend, Full Stack, DevOps, Data Science, etc.)
- 💳 **Premium Subscriptions**: Fully integrated with Razorpay for secure payments and feature limits.

## 🛠️ Tech Stack

### Frontend
- **React 19** - UI library
- **React Router v7** - Client-side routing
- **Vite** - Build tool and dev server
- **TailwindCSS** - Utility-first CSS framework
- **Framer Motion** - Animation library

### Backend
- **Node.js** - Runtime environment
- **Express.js** - Web framework
- **MongoDB** - Database
- **Mongoose** - ODM for MongoDB
- **JWT** - Authentication & Authorization
- **bcryptjs** - Password hashing
- **OpenAI SDK** - Used to interface with the AI models
- **OpenRouter API** - Orchestrates multiple LLM models (Gemini, Qwen, etc.)

## 🚀 Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** (v16 or higher)
- **MongoDB Account** (or local instance)
- **OpenRouter API Key** - [Get it here](https://openrouter.ai)
- **Razorpay Account** (for payment integration)

### Installation

#### 1. Clone the Repository

```bash
git clone https://github.com/chintankasundra/Interv-ai.git
cd Interv.ai
```

#### 2. Backend Setup

```bash
# Navigate to backend directory
cd backend

# Install dependencies
npm install
```

Create a `.env` file in the `backend` directory:

```env
PORT=8000
NODE_ENV=development
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret_key
RAZORPAY_KEY_ID=your_razorpay_key_id
RAZORPAY_KEY_SECRET=your_razorpay_key_secret
OPENROUTER_API_KEY=your_openrouter_api_key
MODEL=google/gemini-2.0-flash-exp:free
```

#### 3. Frontend Setup

```bash
# Navigate to frontend directory
cd ../frontend

# Install dependencies
npm install
```

Update `frontend/src/utils/apiPaths.js` for local development if necessary:

```javascript
export const BASE_URL = "http://localhost:8000"
```

#### 4. Start Development Servers

**Terminal 1 - Backend:**
```bash
cd backend
npm run dev
```
Backend will start on `http://localhost:8000`

**Terminal 2 - Frontend:**
```bash
cd frontend
npm run dev
```
Frontend will start on `http://localhost:5173`

## 🌐 Core API Endpoints

### Authentication
- `POST /api/auth/register` - Register new user
- `POST /api/auth/login` - Login user
- `GET /api/auth/profile` - Get user profile

### Sessions
- `POST /api/sessions/create` - Create interview session
- `GET /api/sessions/my-sessions` - Get all user sessions
- `DELETE /api/sessions/:id` - Delete session

### Payments
- `POST /api/payments/create-order` - Create Razorpay order
- `POST /api/payments/verify-payment` - Verify HMAC SHA256 payment signature
- `GET /api/payments/subscription-status` - Get user plan details

### AI Generation
- `POST /api/ai/generate-questions` - Generate interview questions (Accepts `multipart/form-data` for resume PDFs)
- `POST /api/ai/generate-explanation` - Get concept explanation
